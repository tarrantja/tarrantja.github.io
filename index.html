Hypertext Markup Language (HTML)

Purpose

HTML is the core mark-up language of the World Wide Web. It's purpose is to describe the structure of the text-based content of a website (W3Schools, 2020).

Description

Content to be displayed is arranged into a series of elements, represented by tags. These tags are labelled pieces of content, such as headings, paragraphs and tables. Hover over each of the lines of HTML below to reveal where these elements exist on this page.


    <body>  <!--This opens the entire body of the page--> 
        <ul>...</ul> <!--Unordered List-->
        <h1>This is a Heading 1</h1>
        <h2>This is a Heading 2</h2>
        <p>This is a paragraph</p>
        <table>...</table> 
    </body>

HTML pages can be linked directly to other documents or pages, creating a 'web' of interlinked resources. Typically this is done using the anchor tag as shown below. This has been done in the navigation bar to link the four pages of this website.


<a href="location_of_different_page"> Name of Link </a>

History

Date Event
1989 While working at CERN, Tim Berners-Lee theorises a way for researchers to collaborate remotely by creating a pool of research documents that could be downloaded as computer files (Regget et al, 1998). More so, the text in the documents could be linked directly to other documents with relevant text or diagrams to create a 'web'. The Web, with HTML as its publishing language, was invented.
1990 Sir Tim Berners-Lee creates HTML as a simple language to implement global hypertext links, and encourages dicsussion and open collaboration (Regget et al, 1998). The language was strongly based on SGML, a method for marking up text into structural units. Similarities include the use of pairs-of-tags and elements such as paragraphs, headings and ordered and unordered lists. Tim also creates the world's first web browser, the WorldWideWeb program.
1992 Joseph Hardin and Dave Thompson from NSCA learn of Tim's invention and, realising the importance, develop a browser of their own, Mosaic (Regget et al, 1998).
1993 Dave Ragget develops the Arena browser. HTML has fast become a democratic language where developers implement new features in their browsers, and the web community would decide if they stayed (Veen, 1997).
1994 The specification for HTML 2 is formed by Tim Berners-Lee and David Connolly, adding new features to the original HTML specifications (Shannon, 2012).
1995 HTML 3 is published by Dave Raggett to capture the broad range of HTML tags in use, as well as a range of new tags. Although the draft was comprehensive, it was believed to be too large and full of new proposals to implement. Ultimately, HTML 3 was abandoned (Regget et al, 1998).
1997 HTML 3.2 is endorsed by the W3 Consortium as a HTML specification accepted by most key stakeholders, including the major broswer vendors (Regget et al, 1998). This is followed by the release of HTML 4 later the same year (W3C, 2014).
1998 to 2003 The evolution of HTML is put on hold while W3C work on XHTML.
2004 Browsers such as Opera, Mozilla and Apple form WHATWG, a working group to continue evolving HTML (W3C, 2014).
2007 The HTML 5 specification is published under a W3C copyright, the 5th and most recent major revision of the core language (W3C, 2014). A less restrictive version continues to be worked on by WHATWG, termed the Living Standard for HTML.
