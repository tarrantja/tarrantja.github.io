Cascading Style Sheets (CSS)

Purpose

CSS is a method for managing a site's overall style through a separate stylesheet. CSS allows styling to be separated from content (written in HTML) and applied consistently across the whole website. The term cascading comes from the ability to combine multiple CSS files to determine the style for one page. This means that styles can inherit and overwrite previously declared styles (Hoffman, 2017).

Description

A CSS rule set is described in two parts. First, the selector such as an element (paragraph, headings, etc), class or id. Second, a series of declarations that assign values to style properties such as text colours, background colour, font and margins. The syntax is as follows:


selector {property1: value1; property2: value2;}

History

Date Event
1994 As the Web starts to be used as a publishing platform, CERN employee Hakon Wium Lie identifies the need to style pages. This functionality was not provided by HTML at the time. Hakon publishes the first draft of the Cascading HTML Style Sheets proposal (Bos, 2016), providing a useful basis for discussion. CSS was distinguished from other styling alternatives by accounting for the design requirements of both author and user, acknowledging they would need to be ‘cascaded’ in some way.
1995 The www-style mailing list is created. This would greatly shape the direction of CSS (CSS Class, 2016). Debate over the adoption of CSS focussed on the novel idea of balancing author and user requirements, as well as its simplicity (Bos, 2016). Some argued that a full programming language was required to style sheets.
1996 Internet Explorer becomes the first browser to support CCS, followed by Netscape (Bos, 2016). CSS level 1 emerges as a W3C recommendation later that year.
1997 Cascading Style Sheets and Formatting Properties Working Group forms within W3C to work on additional features.
1998 CSS level 2 becomes the W3C recommendation.
2011 CSS 3 begins to be released, with capabilities split into modules. CSS 4 follows suit and is released to build on CSS 3 modules.
